Use CSS Reset to improve browser compatability

CSS Reset[1] is a widely used practice whereby developers reset the styles of all elements to a baseline value before applying their own CSS style to a website. This is because every browser has its own default "user agent stylesheet" used to make unstyled websites more legible. By using a CSS Reset, you can ensure that your CSS styles will look the same regardless of a user's browser.
